## Description
Smart Habit Tracker is a full-featured habit tracking application that helps users build positive habits through daily tracking, streak counters, progress visualization, and reminders. The app uses localStorage for data persistence, Chart.js for visual analytics, and responsive CSS for mobile compatibility.
